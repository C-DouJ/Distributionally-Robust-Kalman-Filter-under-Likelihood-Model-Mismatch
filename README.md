# Distributionally-Robust-Kalman-Filter-under-Likelihood-Model-Mismatch
This repository provides the implementation of the algorithms proposed in the authors’ two papers published in IEEE Transactions on Aerospace and Electronic Systems (TAES), referred to as Part I and Part II.

The current version includes the complete implementation of the algorithm presented in Part I.
The code corresponding to Part II is still under preparation and will be released soon.
